# Farmer-to-Consumer Marketplace Management System

## Project Description

This project is a Core Java based Farmer-to-Consumer Marketplace Management System developed during the Upskill Campus Full Stack Development Internship.

### Features

- Add Farmer
- Add Product
- Search Product
- Buy Product
- Order History
- Inventory Management
- Farmer Earnings Dashboard

### Technologies Used

- Java
- OOP Concepts
- ArrayList
- Scanner Class
